## Adding a New Scraper

To add a new scraper to the registry:

1. Use the scraper template: `templates/scraper-template/`
2. Run the creation script: `scripts/create-scraper-repo.sh [country] [portal]`
3. The registry will be automatically updated

See [CONTRIBUTING.md](../docs/CONTRIBUTING.md) for detailed instructions.
